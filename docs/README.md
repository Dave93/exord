# Документация системы EXORD

## Регламенты работы с функционалом списания продуктов

В этой папке находятся регламенты по использованию функционала списания продуктов в системе EXORD.

### Доступные документы

1. **writeoff-regulation-ru.pdf** - Регламент на русском языке
2. **writeoff-regulation-uz.pdf** - Регламент на узбекском языке

### Основные положения

**ВАЖНО!** С момента внедрения данного функционала:
- Все списания продуктов должны производиться **ТОЛЬКО через систему EXORD**
- Запрещается создавать акты списания напрямую в системе iiko
- После утверждения офисом, акт списания автоматически создается в iiko

### Целевая аудитория

Регламенты предназначены для:
- Поваров, барменов, кондитеров (создание списаний)
- Менеджеров точек (контроль списаний)
- Администраторов и офис-менеджеров (утверждение списаний)

### Процесс работы

#### Для сотрудников точек:
1. Создайте списание через систему EXORD
2. Прикрепите фотографии списываемых продуктов (обязательно!)
3. Укажите причину списания в комментарии
4. Дождитесь утверждения офисом

#### Для офиса:
1. Проверьте списание (фото, количество, причина)
2. Скорректируйте количество при необходимости
3. Утвердите списание
4. Система автоматически создаст акт в iiko

### Интеграция с iiko

После утверждения списания:
- Система автоматически формирует документ списания
- Отправляет данные в систему iiko через REST API v2
- Создает акт списания в iiko со статусом "NEW"
- Передает все утвержденные позиции и количества

### Регенерация PDF документов

Если необходимо обновить PDF документы из исходных Markdown файлов, выполните:

```bash
cd docs
python3 generate_pdf_simple.py
```

### Исходные файлы

- `writeoff-regulation-ru.md` - Исходный регламент на русском языке (Markdown)
- `writeoff-regulation-uz.md` - Исходный регламент на узбекском языке (Markdown)
- `generate_pdf_simple.py` - Python скрипт для генерации PDF из Markdown

### Требования для генерации PDF

Для запуска скрипта генерации необходимо установить:

```bash
pip install reportlab markdown
```
